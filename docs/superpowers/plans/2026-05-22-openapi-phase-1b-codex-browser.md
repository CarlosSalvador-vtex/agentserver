# OpenAPI Phase 1.b — Codex Browser Sessions tag Implementation Plan

**Goal:** Annotate 1 Codex Browser Sessions REST endpoint at
`/api/workspaces/{wid}/browsers`. The handler is fully implemented in
`internal/server/codex_browsers.go`; it uses an unexported local struct
(`codexBrowser`) that must be promoted to `api_types.go`.

**Prereq:** Stacked on `feat/openapi-phase-1b-codex-tokens` (PR #156).

**Reference spec:** Same conventions as PR #152–#156.

---

## Recon findings

Only **1 user-facing REST endpoint** exists for Codex Browser Sessions:

| Method | Path | Handler | Auth | Notes |
|---|---|---|---|---|
| GET | `/api/workspaces/{wid}/browsers` | `handleListCodexBrowsers` | cookie + workspace member | Returns tokens annotated with live session info (online status, client IP, etc.) |

The internal session lifecycle endpoints (`/api/internal/codex/tokens/session-open`,
`session-close`, `session-update`) are service-to-service RPC — **out of scope** per
spec. No create/revoke browser-session REST endpoints exist; those actions are
performed via Codex Tokens (PR #156). No WebSocket endpoints to annotate here.

---

## File structure

**Modify:**

- `web/src/lib/api.ts` — alias `CodexBrowser` to the generated schema; migrate

**No new files needed** — direct annotation pattern (handler is self-contained,
not proxied).

## Task breakdown

### Task 1: DTOs — `internal/server/api_types.go`

Append after `// --- Codex Tokens ---` block:

```
// --- Codex Browser Sessions ---

// CodexBrowserItem is one entry returned by GET /api/workspaces/{wid}/browsers.
// It mirrors RemoteExecutor so DeviceListPanel can render both without branches.
// is_online is true when at least one open browser session exists for the token.
type CodexBrowserItem struct {
    ID             string     `json:"id" validate:"required"`
    Name           string     `json:"name" validate:"required"`
    WorkspaceID    string     `json:"workspace_id" validate:"required"`
    CreatedAt      time.Time  `json:"created_at" validate:"required"`
    ExpiresAt      time.Time  `json:"expires_at" validate:"required"`
    IsOnline       bool       `json:"is_online"`
    LastUsedAt     *time.Time `json:"last_used_at,omitempty" extensions:"x-nullable=true"`
    ClientIP       string     `json:"client_ip,omitempty"`
    ClientUA       string     `json:"client_ua,omitempty"`
    CodexVersion   string     `json:"codex_version,omitempty"`
    OS             string     `json:"os,omitempty"`
    ConnectedAt    *time.Time `json:"connected_at,omitempty" extensions:"x-nullable=true"`
    DisconnectedAt *time.Time `json:"disconnected_at,omitempty" extensions:"x-nullable=true"`
} // @name CodexBrowserItem
```
